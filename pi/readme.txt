Bare metal samples working on Raspberry PI B

mallocXX - malloc samples
printf01 - standalone printf sample (needs uart and __aeabi_uidiv*.*)
sdchb01XX  - sd card access lib extracted from moizumi99/RPiHaribote
sdchh01XX  - sd card access lib extracted from hldswrth forum zip file
sdcrp01XX  - sd card access lib extracted from jncronin/rpi-boot
uartXX   - simple uart (abs addresses)
